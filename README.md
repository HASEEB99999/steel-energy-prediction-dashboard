# ⚡ Steel Industry Energy Prediction Dashboard

## 📋 Project Overview

This is a FastAPI web dashboard for predicting energy consumption in steel manufacturing using Machine Learning.

### Features
- 📊 EDA Dashboard with 3 visualizations
- 🔮 Real-time energy consumption prediction
- 🏭 PCA dimensionality reduction (18 → 7 features)
- ✅ Model performance: RMSE 7.51 kWh, R² 0.862
